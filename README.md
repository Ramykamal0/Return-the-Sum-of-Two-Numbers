# Return the Sum of Two Numbers

## 📝 Description
A simple C# console application that takes two numbers as input and returns their sum.

## 🛠️ Language & Tools
- Language: C#
- IDE: Visual Studio
- Framework: .NET

## 🚀 How to Run
1. Clone the repository
2. Open `Return the Sum of Two Numbers.slnx` in Visual Studio
3. Press `F5` to run
